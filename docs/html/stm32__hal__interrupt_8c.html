UART_HandleTypeDef * huart )

Forward HAL_UART_RxCpltCallback() into uartbinlib.

The function feeds the received byte, then immediately arms the next byte. If re-arming fails, it resets the protocol parser so no partial frame leaks into the next successful receive operation.

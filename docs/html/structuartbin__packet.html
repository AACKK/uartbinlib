Detailed Description

CRC-checked packet delivered to the application.

The payload pointer aliases uartbin_config::rx_payload_buffer. It is valid only until the next call to uartbin_feed(), uartbin_feed_at(), uartbin_feed_byte(), uartbin_feed_byte_at(), uartbin_poll(), or uartbin_reset() on the same context.

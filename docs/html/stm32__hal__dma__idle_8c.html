STM32 HAL DMA + idle-line RX ornegi.

STM32 HAL interrupt RX ornegi.

HAL_UARTEx_ReceiveToIdle_DMA kullanir ve alinan her blogu uartbin'e besler. TX, blocking HAL_UART_Transmit kullanir; cunku uartbin_send tek cerceve icin write hook'u birkac kez cagirir. Non-blocking transmit gerekiyorsa bunu kendi TX queue yapin ile degistir.

Gercek projede cihaz HAL header'ini include et, ornegin:

#include "stm32f4xx_hal.h"

TX notu: uartbin_send tek cerceve icin write hook'u birden fazla kez cagirir. Write hook donmeden once yazmayi bitirmeli veya byte'lari TX kuyruguna kopyalamalidir. Bu ornek bu nedenle blocking HAL_UART_Transmit kullanir.
